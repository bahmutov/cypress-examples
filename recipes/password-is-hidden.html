# Password Is Hidden

<div class="form-row">
  <div class="form-group">
    <label for="password">Password</label>
    <input
      type="password"
      class="form-input"
      name="my-secret"
      id="my-secret"
      value=""
    />
  </div>
</div>

Confirm the browser hides (opens new window) the entered characters by checking that the element is an <input type="password">. We can check the element using two different assertions.

cy.get('#my-secret')
  .should('have.prop', 'nodeName', 'INPUT')
  .and('have.attr', 'type', 'password')

We could also include the node name and the password attribute in our query. We can also hide what we type into the input element, see Keep passwords secret in E2E tests (opens new window).

cy.get('input#my-secret[type=password]').type('123!secret', {
  log: false,
})
